# あにまん掲示板 レスURLコピーツール
## PC版Chromeへのインストール方法:  
緑色のCodeボタンをクリックし、開いたメニューの一番下「Download Zip」でダウンロードし、適当なフォルダに解凍してください。  
拡張機能の管理画面で「デベロッパーモード」を有効にしてください。  
「パッケージ化されていない拡張機能を読み込む」で解凍したファイルのフォルダを選択するとインストールされます。  
スレッドを新たに開くかリロードし、レスのどこか（**ヘッダー、本文、画像のどこでもOK**）で右クリックし、「このレスのURLをコピー」を選択するとコピーされます。  
  
コードが信頼できない場合はAIにファイル投げつけて「ホントにこれ大丈夫なやつ？」って聞いてみてください。
